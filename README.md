# THE PANTRY — 인공감미료 기준표

더 팬트리 내부 기준 문서. 성분표에서 인공감미료 이름이 보이면 바로 확인할 수 있는 검색 가능한 레퍼런스.

## 배포 방법 (GitHub Pages)

1. 이 레포지토리를 `thepantryjisoo` 계정에 생성
2. `index.html`, `manifest.json`, `sw.js` 업로드
3. **Settings → Pages → Branch: main → Save**
4. 약 1분 후 링크 생성됨:  
   `https://thepantryjisoo.github.io/[레포이름]/`

## 아이폰 앱으로 설치

1. Safari에서 링크 열기
2. 하단 공유 버튼 탭
3. **홈 화면에 추가** 선택
4. 앱처럼 실행됨 (오프라인도 가능)

## 파일 구조

```
index.html    — 메인 페이지 (검색 + 필터 + 전체 데이터)
manifest.json — PWA 앱 정보
sw.js         — 오프라인 캐싱
```

## 업데이트

`index.html` 안의 `DATA` 배열에 항목 추가/수정 후 GitHub에 다시 업로드하면 즉시 반영됩니다.
